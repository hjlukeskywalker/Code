변수선언하기

변수선언하는 방법 var 변수명;
자바스크립트에서 변수를 선언할때는 자료형을 지정하지 않음 자바스크립트 변수의 타입은 값을 대입했을때 그 값의 타입으로 결정된다.

변수는 접근할 수 있는 범위에 따라 영역(scope)이 구분
global scope : page내 어디서든 접근이 가능한 변수 -> window객체의 멤버변수 local scope : function내부에서 선언한 변수 -> function내부에서만 사용이 가능

자바스크립트는 블록범위를 허용하지 않는다. 자바에서는..... if(){ String test } 외부에서 접근이 가능? for(){int a} a -> 불가능 자바스크립트! if(){var test} console.log(a)-> 외부에서 접근이 가능함.

변수를 선언시에 주의할점! 변수를 중복으로 선언하면 에러가 발생하지 않음! 변수중복에 대해 잘 체킹을 해야했음. -> es6 변경!

es6에 와서 변수를 선언하는 예약어를 추가 블록스코프를 유지하고, 중복선언하는것을 방지하는 기능 let, const예약어를 사용해서 변수를 선언! let : 블록스코프영역을 가지는 변수를 선언할때 const : 상수를 저장하는 변수를 선언할때 사용

자바스크립트 자료형

자바스크립트는 자료형을 가지고 있으며 변수의 자료형이 결정되는 것은 변수에 값(리터럴) 대입할때 임. 각 자료형을 알아보고, 리터럴표현방법을 알아보자. 문자열(string), 숫자(number), 배열([]), 객체({}), 함수(function()), undefined, boolean
