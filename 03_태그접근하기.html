태그접근하는 방법

javascript에서 태그는 하나의 객체로 본다.(DOM) 객체에서는 태그를 표현하는 properties(속성)가 있음
태그에 접근하는 방법은 4가지가 있음 * 속성값을 이용해서 태그를 가져옴

  1. id값을 기준으로 가져오기
  2. class값을 기준으로 가져오기
  3. tagName값을 기준으로 가져오기
  4. name값을 기준으로 가져오기

id값을 기준으로 가져오기

태그의 id속성에 있는 값을 기준으로 태그를 가져옴 document.getElementById("id속성 값"); -> id속성의 값이 일치하는 태그 한개를 가져옴.

선택자를 이용해서 태그를 가져오기

css에서 사용한 선택자를 이용해서 태그를 지정하여 가져오는 것 #test -> id가 test인 태그 .test -> class가 test인 태그 [type=test] -> type속성이 test인 태그 div>p -> div태그에 직접 연결된 p
document.querySelector("선택자"); -> 한개의 태그만 가져오는것 다수가 있으면 첫번째 태그만 가져옴 document.querySelectorAll("선택자"); -> 다수의 태그를 가져오는 것
