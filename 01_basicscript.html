자바스크립트 이용하기

자바스크립트로 페이지를 interactive하게 구성하려면 자바스크립트 구문을 태그내 속성(inline), script태그내부, 외부js파일에 작성한다.

  1. inline : 태그내부의 이벤트속성(onOOO)에 직접 javascript코드를 작성하는 것 * 간단한 구문만 사용
  2. internal : 페이지의 script태그안에 javascript코드를 작성하는것
  3. external : ooo.js 파일에 javascript코드를 작성하고 페이지에서 script태그를 이용해서 불러와(script src="주소") 사용 / 서버내부에 저장된 파일이나 외부(https...주소)에서 지원하는 파일 두개다 가능
