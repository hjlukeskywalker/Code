background-color

태그마다 페이지에서 차지하는 각 영역이 있는데 그 영역에 대한 색상을 선택할 수 있게 하는 속성

  • html
  • css
  • javascipt
  • jquery

background-clip

태그(element)가 차지하고 있는 영역에 대한 범위 border-box : border의 범위까지 배경으로 본다. padding-box : padding의 범위까지 content-box : content의 범위까지

border-box
padding-box
content-box

background-image

태그에 이미지를 넣을때 사용하는 속성 사진으로 불러와서 출력 url("경로")

background-repeat

기본적으로 배경이미지를 설정하면 반복(바둑판)으로 출력이 됨 조정할 수 있는 속성. repeat, repeat-x/repeat-y, no-repeat

background-size

배경이미지의 크기를 조정할 수 있는 속성 contain : 이미지가 영역안에 모두 들어갈 수 있게 확대, 축소 -> 이미지가 모두 출력 cover : 영역이 꽉차게 설정함 -> 이미지 일부만 출력 숫자/백분률 : 크기를 지정

background-position

배경이미지의 위치를 조정하는 속성 background-potision : left||right||top||bottom||center||숫자(절대,상대)

background-origin

배경이미지가 차지하는 범위를 설정하는 속성 background-origin : border-box,padding-box,content-box;

background-attachment

페이지가 스크롤에 의해 움직일때 배경 이미지의 위치를 설정하는 속성 scroll : 페이지가 움직이면 따라서 움직이는 속성 -> 페이지 이동하면 안보임. fixed : 페이지가 움직여도 설정한 위치에 있는 속성 -> 페이지이동해도 계속 보임.

linear-gradient

색상을 점직적으로 표현해주는 속성 background:linaer-gradient(각도||방향,시작색, 끝색)
